# Diagramas — VPS Hermes

Repo de diagramas de arquitectura, flujos y modelos. Editables en [app.diagrams.net](https://app.diagrams.net) conectando a este repo vía GitHub.

## Estructura

```
equipo/         # Arquitectura del equipo (Victoria, Coder, BDD)
turnitoslr/     # Diagramas específicos de TurnitosLR
hierronort/     # Diagramas específicos de HIERRONORT
_index/         # Lienzo principal con links a los otros
```

## Cómo editar

1. Andá a https://app.diagrams.net
2. Elegí **GitHub** como storage
3. Logueate con la cuenta `Fqui`
4. Abrí este repo (`diagramas`)
5. Elegí el archivo `.drawio` que querés editar
6. Los cambios se commitean automáticamente cuando guardás

## Cómo pedir un diagrama nuevo

Pedile a Victoria (Hermes) por chat:
- "Necesito un diagrama de [X]"
- O directamente: "Generame un .drawio de [Y]"

Victoria commitea al repo. Vos lo abrís y editás si querés.
